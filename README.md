# Olist E-Commerce Analysis

Exploratory data analysis of the Brazilian E-Commerce Public Dataset by Olist using Python and pandas.

To be defined — currently in exploration phase.

## Tools

- Python 3.12
- pandas, numpy
- matplotlib, seaborn
- Jupyter notebooks

## Data

Dataset: [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

Download the CSVs from Kaggle and place them in a `data/` folder at the project root.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
